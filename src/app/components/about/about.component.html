Informacion

La API de Marvel Comics es un gran recurso para desarrolladores principiantes y avanzados, ya que abre la puerta para aprender a administrar grandes cantidades de datos provenientes de una API bellamente construida. Con NgMarvel hemos explorado el nuevo Bootstrap 5, el tema Bootswatch, Angular Infinite Scroll, Font Awesome, operadores y funciones RxJS y muchos más..

Esta aplicación trata sobre la implementación de la aplicación Angular utilizando la API de Marvel Comics como fuente de datos. Los datos utilizados en la aplicación pertenecen a Marvel y se utilizan solo para aprender y explorar los conceptos de Angular y sobre la API de Marvel Comics. Esta aplicación no se utiliza como un producto comercial..
